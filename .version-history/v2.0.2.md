# Version 2.0.2 — 2026-05-27

## Summary
Created an elegant, space-efficient `UserBadge` mobile layout visible only on phone/tablet viewport widths, displaying a high-contrast icon badge alongside concise uppercase tier labels ("ADMIN", "PRO", or "FREE") and a pulsing green dot active indicator. hard-coded bespoke CSS color classes in the UserBadge component for perfect color contrast across mobile and desktop interfaces in dark mode.
