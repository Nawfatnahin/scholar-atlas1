# Version 1.0.2 — 2026-05-24

## Summary
Bespoke dashboard visual overhaul, light mode contrast restoration, and edge environment build optimizations. Redesigned the SemesterProgressWidget to support rounded, floating glassmorphism cards and pulsing radial halo alerts. Replaced generic skeletons with modern glass loaders. Restored high-contrast solid white panel backgrounds for excellent visibility against your cream base background. Repaired about page readability and configured automatic ESLint bypassing to ensure flawless Cloudflare Pages deployments.

## Database Tables (Supabase)
- users (via Supabase Auth)
- subjects (id, user_id, name, color, created_at, course_code, semester_start_date, total_weeks)
- attendance_logs (id, subject_id, status, date)
- attendance_records (computed/referenced in UI)
- class_sessions (id, subject_id, user_id, date, status, is_extra)
- tasks (id, user_id, subject_id, title, due_date, status, priority)
- cgpa_settings (id, user_id, target_cgpa)
- cgpa_entries (id, user_id, cgpa, created_at)
- cgpa_courses_manual / cgpa_courses_auto (per subject mark tracking)
- holidays (id, user_id, name, date)
